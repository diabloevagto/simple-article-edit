# simple-article-edit

實做 CRUD 線上文章編輯功能，並且利用 firebase 實現 CRUD 功能

# 使用步驟
1. 輸入自己希望的使用者名稱，之後的文章變更皆在此名稱底下，此部分不做驗證
2. 可以自行新增文章，給予當下時間做唯一 key 值
3. 點選時間進入文章內可預覽以及編輯
  * 預覽為實際 firebase 儲存的內容
  * 編輯區為目前修改暫存內容，按下更新才會實際送出結果
4. 點選文章列表的 X 可刪除文章

# 使用 lib
* vue
* vue-router
* vuex
* element-ui
* axios
* firebase
